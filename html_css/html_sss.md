#HTML

##HTML Introduction

- HTML : Hyper Text Markup Language

  - Hyper Text : 하이퍼링크로 연결된 웹 문서 => 웹 페이지
  - Markup Language : 표시 언어

- HTML 표시 내용
  - Web page의 Contents (내용 표시)
  - Web Page의 Structure (구조 설명)

## HTML Basic

-기본구조

※ ` : backtick

```
<!DOCTYPE html>
<html>
  <head>
    웹 문서의 부가정보
  </head>
  <body>
    웹 문서의 내용
  </body>
</html>
```
